## Introduction

Manually writing IPv6 router configurations leads to inconsistencies and errors. Jinja2 templates with YAML data sources enable DRY (Don't Repeat Yourself) configuration management. Define your network data once in YAML, then render consistent configurations for any router platform.

```yaml

routers:
  - name: R1
    loopback: "5f00:fe81:1::/128"
    asn: 65001
    interfaces:
      - name: GigabitEthernet0/0/0/0
        description: "Link to R2"
        ipv6_address: "fd00:12::1/64"
        ospfv3_cost: 10
      - name: GigabitEthernet0/0/0/1
        description: "Link to R3"
        ipv6_address: "fd00:13::1/64"
        ospfv3_cost: 20
    bgp_neighbors:
      - address: "5f00:fe81:2::"
        remote_as: 65001
        description: "R2 iBGP"
      - address: "5f00:fe81:3::"
        remote_as: 65001
        description: "R3 iBGP"
```

## Cisco IOS-XR Interface Template

```jinja2
{# templates/ios_xr_interface.j2 #}
{# Render IOS-XR interface config from router data #}

! === Router: {{ router.name }} ===
! Generated by Jinja2 automation

interface Loopback0
 description "{{ router.name }} Loopback"
 ipv6 address {{ router.loopback }}
!

{% for iface in router.interfaces %}
interface {{ iface.name }}
 description "{{ iface.description }}"
 ipv6 address {{ iface.ipv6_address }}
 ipv6 enable
{% if iface.ospfv3_cost is defined %}
 router ospfv3 1 area 0 cost {{ iface.ospfv3_cost }}
{% endif %}
!
{% endfor %}
```

## FRR BGP Template

```jinja2
{# templates/frr_bgp.j2 #}
! FRR BGP configuration for {{ router.name }}

router bgp {{ router.asn }}
  bgp router-id {{ router.loopback | replace('::/128', '') }}
  no bgp ebgp-requires-policy

{% for neighbor in router.bgp_neighbors %}
  ! {{ neighbor.description }}
  neighbor {{ neighbor.address }} remote-as {{ neighbor.remote_as }}
  neighbor {{ neighbor.address }} update-source Loopback0
  {% if neighbor.remote_as == router.asn %}
  neighbor {{ neighbor.address }} next-hop-self
  {% endif %}
{% endfor %}

  address-family ipv6 unicast
{% for neighbor in router.bgp_neighbors %}
    neighbor {{ neighbor.address }} activate
{% endfor %}
    network {{ router.loopback | replace('/128', '/48') }}
  !
!
```

## Python Rendering Script

```python
#!/usr/bin/env python3
import yaml
import os
from jinja2 import Environment, FileSystemLoader

def render_configs(data_file: str, template_dir: str, output_dir: str):
    """Render configurations for all routers."""
    env = Environment(
        loader=FileSystemLoader(template_dir),
        trim_blocks=True,
        lstrip_blocks=True,
    )

    with open(data_file) as f:
        data = yaml.safe_load(f)

    os.makedirs(output_dir, exist_ok=True)

    for router in data["routers"]:
        # Render interface config
        tpl = env.get_template("ios_xr_interface.j2")
        output = tpl.render(router=router)
        out_file = f"{output_dir}/{router['name']}_interfaces.conf"
        with open(out_file, "w") as f:
            f.write(output)
        print(f"Written: {out_file}")

        # Render BGP config
        tpl = env.get_template("frr_bgp.j2")
        output = tpl.render(router=router)
        out_file = f"{output_dir}/{router['name']}_bgp.conf"
        with open(out_file, "w") as f:
            f.write(output)
        print(f"Written: {out_file}")

if __name__ == "__main__":
    render_configs(
        data_file="data/routers.yml",
        template_dir="templates",
        output_dir="output",
    )
```

```jinja2
#!/usr/sbin/nft -f
# Generated for: {{ router.name }}

flush ruleset

table inet filter {
  chain input {
    type filter hook input priority 0; policy drop;

    # Accept loopback
    iifname "lo" accept

    # Accept established/related
    ct state established,related accept

    # Accept ICMPv6 (required for IPv6 operation)
    ip6 nexthdr icmpv6 accept

    # Accept SSH from management subnet
{% for mgmt_net in management_subnets %}
    ip6 saddr {{ mgmt_net }} tcp dport 22 accept
{% endfor %}

    log prefix "DROP: " drop
  }

  chain forward {
    type filter hook forward priority 0; policy drop;
{% for iface in router.interfaces %}
    iifname "{{ iface.name }}" accept comment "{{ iface.description }}"
{% endfor %}
  }
}
```
