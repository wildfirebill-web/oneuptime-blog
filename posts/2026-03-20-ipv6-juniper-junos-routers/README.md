# How to Configure IPv6 on Juniper Junos Routers

Author: [nawazdhandala](https://www.github.com/nawazdhandala)

Tags: IPv6, Juniper, Junos, Router, BGP, Networking

Description: Configure IPv6 addressing, static routing, OSPF, and BGP on Juniper Junos routers with family inet6 configuration and verification commands.

## Introduction

Juniper Junos uses an address-family concept where IPv4 is `family inet` and IPv6 is `family inet6`. This clean separation makes it easy to configure IPv6 alongside IPv4 using familiar Junos structures.

## Basic Interface Configuration

```text
# Configure IPv6 address on an interface using set commands

set interfaces ge-0/0/0 unit 0 family inet6 address 2001:db8:1:1::1/64
set interfaces ge-0/0/0 unit 0 description "LAN Interface"

# Assign a link-local address explicitly (optional - auto-generated by default)

set interfaces ge-0/0/0 unit 0 family inet6 address fe80::1/64

# Configure loopback for router-id purposes
set interfaces lo0 unit 0 family inet6 address 2001:db8::1/128

# Commit and verify
commit
show interfaces ge-0/0/0 detail | grep inet6
```

## Configuring Static IPv6 Routes

```text
# Static route to a destination prefix
set routing-options rib inet6.0 static route 2001:db8:2::/48 next-hop 2001:db8:0:1::2

# Default IPv6 route
set routing-options rib inet6.0 static route ::/0 next-hop 2001:db8:isp::1

# Floating static route (higher preference number = lower preference)
set routing-options rib inet6.0 static route ::/0 next-hop 2001:db8:isp2::1 preference 200
```

## Configuring OSPFv3

```text
# Configure OSPFv3 (IPv6 OSPF uses version 3)
set protocols ospf3 area 0.0.0.0 interface ge-0/0/0.0
set protocols ospf3 area 0.0.0.0 interface ge-0/0/1.0
set protocols ospf3 area 0.0.0.0 interface lo0.0 passive

# Set router-id explicitly
set routing-options router-id 1.1.1.1

# Verify
show ospf3 neighbor
show ospf3 database
```

## Configuring BGP with IPv6

```bash
# Configure BGP group for IPv6 peering
set routing-options autonomous-system 65001
set routing-options rib inet6.0 aggregate route 2001:db8::/48

set protocols bgp group UPSTREAM-IPV6 type external
set protocols bgp group UPSTREAM-IPV6 family inet6 unicast
set protocols bgp group UPSTREAM-IPV6 peer-as 65000
set protocols bgp group UPSTREAM-IPV6 neighbor 2001:db8:isp::1

# Export policy to announce your prefix
set policy-options policy-statement EXPORT-IPV6 term 1 from family inet6
set policy-options policy-statement EXPORT-IPV6 term 1 from route-filter 2001:db8::/48 orlonger
set policy-options policy-statement EXPORT-IPV6 term 1 then accept

set protocols bgp group UPSTREAM-IPV6 export EXPORT-IPV6
```

## Configuring Firewall Filters (ACL)

```text
# Create an IPv6 firewall filter
set firewall family inet6 filter IPV6-PROTECT term ALLOW-ICMPV6 from protocol icmp6
set firewall family inet6 filter IPV6-PROTECT term ALLOW-ICMPV6 then accept

set firewall family inet6 filter IPV6-PROTECT term ALLOW-ESTABLISHED from protocol tcp
set firewall family inet6 filter IPV6-PROTECT term ALLOW-ESTABLISHED from tcp-established
set firewall family inet6 filter IPV6-PROTECT term ALLOW-ESTABLISHED then accept

set firewall family inet6 filter IPV6-PROTECT term DENY-REST then discard

# Apply filter to interface (inbound)
set interfaces ge-0/0/0 unit 0 family inet6 filter input IPV6-PROTECT
```

## Full Configuration in Stanza Format

```text
interfaces {
    ge-0/0/0 {
        unit 0 {
            description "LAN";
            family inet6 {
                address 2001:db8:1:1::1/64;
            }
        }
    }
    lo0 {
        unit 0 {
            family inet6 {
                address 2001:db8::1/128;
            }
        }
    }
}

routing-options {
    router-id 1.1.1.1;
    autonomous-system 65001;
    rib inet6.0 {
        static {
            route ::/0 next-hop 2001:db8:isp::1;
        }
    }
}

protocols {
    ospf3 {
        area 0.0.0.0 {
            interface ge-0/0/0.0;
            interface lo0.0 { passive; }
        }
    }
}
```

## Verification Commands

```text
# Show IPv6 addresses on all interfaces
show interfaces terse | match inet6

# Show IPv6 routing table
show route table inet6.0

# Show BGP summary for IPv6
show bgp summary

# Show OSPFv3 neighbor table
show ospf3 neighbor

# Show IPv6 neighbor discovery cache
show ipv6 neighbors

# Ping test with IPv6
ping inet6 2606:4700:4700::1111 count 5
traceroute inet6 2606:4700:4700::1111
```

## Conclusion

Juniper Junos IPv6 configuration centers on the `family inet6` addressing and the `inet6.0` routing table. The hierarchical configuration model makes it easy to audit and manage IPv6 settings separately from IPv4. For internet-connected deployments, pair BGP IPv6 with an aggregate route and export policy to properly announce your IPv6 prefix to upstream providers.
